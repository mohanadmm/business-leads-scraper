# Yellow Pages Business Leads Scraper

A Python scraper that collects business leads from Yellow Pages and exports them to an Excel file.

## Extracted Data
- Business Name
- Category
- Phone
- Address
- Website
- Listing URL

## Technologies Used
- Python
- Selenium
- BeautifulSoup
- Pandas
- OpenPyXL

## Project Structure

- yp_scraper.py
